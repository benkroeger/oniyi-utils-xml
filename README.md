> utils for handling xml data


## Install

```sh
$ npm install --save oniyi-utils-xml
```


## Usage

```js
var xmlUtils = require('oniyi-utils-xml');
```

## Functions

* parse
* serialize
* select
* selectUseNamespaces

## Streams

* XMLParseStream
* XMLSerializeStream
